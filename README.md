# AlloServices 🏠  

Welcome to the official documentation for **AlloServices**!  
AlloServices is a mobile application that allows users to book home services like cleaning, plumbing, and electricity from their phone. This guide will help you understand the app’s features, setup, and technologies used.  

---

## 📱 Features  

- 🛠 **Book Home Services** – Browse categories, select sub-services, and place orders.  
- 👤 **Manage Your Account** – Update personal information, profile picture, security settings, and language preferences.  
- 📜 **Order History** – View past orders and reorder quickly.  
- 🌐 **Multilingual Support** – Available in French, Arabic, English, and Chinese.  
- 🔔 **Notifications** – Receive order confirmations and status updates.  
- 📩 **Support Integration** – Send messages to support via Formspree.  
- ☁️ **Image Management** – Profile and service images handled via Cloudinary.  

---

## 🛠️ Technologies Used  

- **Flutter & Dart**  
- **Firebase** – Authentication, Firestore database, Storage, Crashlytics  
- **SharedPreferences** – Local storage for user preferences and profile images  
- **Cloudinary** – Cloud image hosting and optimization  
- **Formspree** – Sending emails without backend  
- **GetX & Provider** – State management and navigation  
- **HTTP & JSON** – Fetch dynamic service and sub-service data  
- **Flutter Intl** – Multilingual support  
- **Git & GitHub** – Version control and collaborative development
